WebSocket based Process Data Access

WebPDA is a protocol for accessing real-time data using WebSocket.

Real-time data is usually related to a process, so it is also called process data. For example, current temperature of a furnace, price of a stock, blood pressure of a person, CPU and memory usage of a server... These variables are called Process Variable (PV) in WebPDA.

Internet of Things, SCADA, finance, health, weather, environment monitoring, measurement and control systems, data center, whenever you need to access your real-time data from the web, WebPDA can ease your life.
